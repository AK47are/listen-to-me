## DTO

### AudioDTO

音频稿件发布请求参数

```json
{
  "title": "Spring Boot 核心原理解析",
  "coverUrl": "https://minio.domain.com/...",
  "description": "深入讲解...",
  "audioUrl": "https://minio.domain.com/...",
  "trialDuration": 30,
  "isPaid": true,
  "price": 19.90,
  "visibility": "PUBLIC"
}
```

字段说明：

- `title`：必填， 标题， 不能为空
- `coverUrl`：必填， 封面临时URL
- `description`：可选， 简介
- `audioUrl`：必填， 音频临时URL
- `trialDuration`：可选， 试听时长（秒）， 0表示无试听， 默认0
- `isPaid`：可选， 是否收费， 默认false
- `price`：isPaid为true时必填， 价格（元）， 必须大于0
- `visibility`：可选， 可见性， 枚举值: PUBLIC / PRIVATE， 默认PUBLIC

正在使用的接口：

1. [发布音频稿件](接口详情文档.md#发布音频稿件)

## VO

### CoinTransactionVO

虚拟币流水视图对象

```json
{
  "id": 1,
  "type": "EXPENSE",
  "amount": 29.90,
  "balanceBefore": 100.00,
  "balanceAfter": 70.10,
  "bizId": "ORD202604060001",
  "bizType": "AUDIO",
  "remark": "购买音频：Spring Boot核心原理解析",
  "createTime": "2026-04-06 10:30:00"
}
```

字段说明：

- `id`：必填， 流水ID
- `type`：必填， 交易类型， 枚举值: EXPENSE(支出) / INCOME(收入)
- `amount`：必填， 变动金额（正数）
- `balanceBefore`：必填， 变动前余额
- `balanceAfter`：必填， 变动后余额
- `bizId`：必填， 业务ID（订单号/充值单号）
- `bizType`：必填， 业务类型， 枚举值: AUDIO / CONSULT / RECHARGE / REFUND
- `remark`：可选， 备注
- `createTime`：必填， 创建时间

正在使用的接口：

1. [分页查询虚拟币流水](接口详情文档.md#分页查询虚拟币流水)


字段说明：

- `id`：必填， 订单ID
- `slotId`：必填， 时间槽ID
- `startTime`：必填， 开始时间， 格式 yyyy-MM-dd HH:mm:ss
- `endTime`：必填， 结束时间， 格式 yyyy-MM-dd HH:mm:ss
- `price`：必填， 预约价格（虚拟币）
- `status`：必填， 订单状态， 枚举值: PENDING_CONFIRM / CONFIRMED / COMPLETED / CANCELLED / REFUND_PENDING / REFUNDED
- `address`：可选， 预约地址， 仅 CONFIRMED / COMPLETED 状态返回
- `userId`：创作者端接口返回， 用户ID， 用于创作者识别预约用户
- `userNickname`：创作者端接口返回， 用户昵称， 用于创作者展示预约用户信息
- `userAvatar`：创作者端接口返回， 用户头像， 用于创作者展示预约用户信息
- `creatorId`：用户端接口返回， 创作者ID， 用于用户识别创作者
- `creatorName`：用户端接口返回， 创作者名称， 用于用户展示创作者信息
- `creatorAvatar`：用户端接口返回， 创作者头像， 用于用户展示创作者信息
- `message`：必填， 用户留言， 用户发起预约时填写
- `rejectReason`：可选， 拒绝原因， 仅退款拒绝时返回， 用于向用户展示拒绝理由
- `createTime`：必填， 创建时间， 格式 yyyy-MM-dd HH:mm:ss
