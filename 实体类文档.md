## PO

### RBAC 权限域

#### SysUser

系统用户表

```json
{
  "id": 1001,
  "username": "zhangsan",
  "password": "$2a$10$...",
  "nickname": "张三",
  "avatar": "https://...",
  "phone": "13800000000",
  "email": "user@example.com",
  "openid": "wx_xxx",
  "isCreator": 0,
  "balance": "100.00",
  "frozenBalance": "0.00",
  "status": "NORMAL",
  "createTime": "2026-04-17 10:00:00",
  "updateTime": "2026-04-17 10:00:00"
}
```

字段说明：

- `id`：必填， 用户ID
- `username`：必填， 账号
- `password`：必填， BCrypt加密密码
- `nickname`：可选， 昵称
- `avatar`：必填， 头像地址
- `phone`：可选， 手机号
- `email`：可选， 邮箱
- `openid`：可选， 三方平台唯一标识
- `isCreator`：必填， 0-听众 1-创作者
- `balance`：必填， 可提现余额
- `frozenBalance`：必填， 账期内冻结金额
- `status`：必填， 状态 NORMAL / BANNED
- `createTime`：必填， 创建时间
- `updateTime`：必填， 更新时间
